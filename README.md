# Probability-scale




# The idea
create a coin flip probability game where if it is 1/2, it will be 1 coin flip to get heads. If it is 1/4, it will be 2 coins / 25%, and lets say you can flip one coin every 3 seconds, it will take 6 seconds to hit them all. So the fun part will be like a game: can you get all the coins to land on heads, or do you lose? The math behind it will be like P = (1/2)^n, and the flips will be visual to realize the outcome. There is also a timer showing how long it will take you to hit it, and fun messages if you hitit .


# Math 

Expected time to win
P= (1/2)^2
T= 3s/(1/2)^2
